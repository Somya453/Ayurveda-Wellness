# 🌿 Ayurveda Wellness Website

A clean and responsive one-page website designed for an Ayurveda wellness center.  
This project focuses on modern UI design, simplicity, and a smooth user experience.

---

## 🔗 Live Demo
https://delicate-alpaca-e695a5.netlify.app/

---

## 🚀 Features
- Responsive design (mobile-friendly)
- Elegant hero section with background image
- Services section with interactive cards
- Clean layout and typography
- Smooth hover effects and basic animations

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- Google Fonts  

---

## 📌 Project Goal
The goal of this project was to create a visually appealing and user-friendly wellness website layout suitable for beginners and frontend practice.
